Returns on Standard \\& Poor's 500 Index
----------------------------------------

Description
~~~~~~~~~~~

daily observations from 1981–01 to 1991–04

*number of observations* : 2783

Usage
~~~~~

::

    data(SP500)

Format
~~~~~~

A dataframe containing :

r500
    daily return S\\&P500 (change in log index)

References
~~~~~~~~~~

Sons,
