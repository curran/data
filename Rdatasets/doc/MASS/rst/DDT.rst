DDT in Kale
-----------

Description
~~~~~~~~~~~

A numeric vector of 15 measurements by different laboratories of the
pesticide DDT in kale, in ppm (parts per million) using the multiple
pesticide residue measurement.

Usage
~~~~~

::

    DDT
